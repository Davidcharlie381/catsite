# Catsite

A basic React site with React Router, styled with Tailwind CSS, consuming <a href="https://thecatapi.com">thecatapi.com</a>.


## Clone locally

- Fork this repo
- clone with

```
git clone https://github.com/yourusername/projectname
```
- Run

```
npm install
```

to install all dependencies.

- Run

```
npm run dev
```

to start the server.
